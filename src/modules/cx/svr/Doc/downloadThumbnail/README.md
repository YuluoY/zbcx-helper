# downloadThumbnail

下载文档缩略图（通过ID）

## 方法签名
```typescript
static downloadThumbnail(params: Object)
```

## 参数说明
| 参数名 | 类型 | 必填 | 说明 |
|--------|------|------|------|
| params | Object | 是 | 参数 |
| ├─ id | Number | 是 | 文档ID |
| ├─ width | Number | 是 | 缩略图宽度 |
| └─ height | Number | 是 | 缩略图高度 |

## 返回值
无

## 功能描述
下载文档缩略图（通过ID）。
