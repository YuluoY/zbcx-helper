# 命令条

`cx-cmdbar` 命令条。

## 基础用法

```vue
<style lang="scss" scoped></style>

<template>
  <cx-cmdbar>
    <cx-button icon="cx-fi ic-department"> 测试 </cx-button>
  </cx-cmdbar>
</template>
```

## 属性

| 名称 | 说明 | 类型 | 可选值 | 默认值 |
| ---- | ---- | ----- | ----- | ----- |
| size | 尺寸 | String | large / default /small | default |

## 插槽

| 名称 | 描述 | 参数 |
| ---- | --- | --- |
| default | 默认插槽 | |
