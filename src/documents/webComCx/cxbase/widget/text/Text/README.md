# 文本

`cx-text` 基于 span 文本。

## 基础用法

```vue
<template>
  <cx-text
    text="示例文本"
    text-size="text1"
  />
</template>
```

## 属性

| 名称 | 说明 | 类型 | 可选值 | 默认值 |
| --- | ---- | ---- | ----- | ----- |
| size | 尺寸 | String | large / default /small | default |
| textSize | 文本大小 | String | text0 / text1 / text2 / text3 / text4 | text1 |
| text | 文本 | String | | |
